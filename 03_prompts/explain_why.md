# Prompt: Why Does This Concept Matter?

> Use this when you're learning something and wondering "when will I ever use this?"
> Fill in the concept name and paste into Cursor chat.

---

I'm learning about: **<concept name here>**

Please explain:

1. **What it is** — in one sentence, using a plain-English analogy (no jargon)
2. **Why it exists** — what problem does it solve? What was the alternative before it?
3. **Where I'll actually use it** — give me a concrete example in the context of my goal track (read `LEARNER_PROFILE.md` first)
4. **A minimal working example** — the simplest possible code that shows it in action (5–10 lines max)
5. **What to watch out for** — one common mistake beginners make with this concept

**Rules:**
- Reference the knowledgebase section if it exists (e.g. "see Stage 3 in `python_first_principles.md`")
- Always give the real-world use case from MY goal track — not a generic example
- Keep it under 300 words total
- If I can try something right now, end with a micro-challenge (1–2 lines to write myself)
