Data Structure	Problem	Suggested Method
Lists	Loop through a list and print each element.	for-loop
Lists	Loop through a list and print elements that are greater than 10.	for-loop + if
Lists	Loop through a list with index and value.	enumerate()
Lists	Loop through a list in reverse order.	reversed()
Lists	Loop through a list and create a new list with squared values.	list comprehension
Strings	Loop through a string and count the vowels.	for-loop + if
Strings	Loop through a string and return a new string with only uppercase characters.	for-loop + isupper()
Strings	Loop through a string and build a dictionary of character frequencies.	for-loop + dict
Strings	Loop through a string backwards and return it.	slicing or reversed()
Strings	Loop through a string and capitalize every other letter.	for-loop + index
Dictionaries	Loop through dictionary keys and print them.	for key in dict
Dictionaries	Loop through dictionary values and sum them.	for val in dict.values()
Dictionaries	Loop through dictionary items and print key-value pairs.	for key, val in dict.items()
Dictionaries	Loop through dictionary and create a list of keys with values over 100.	dict.items() + if
Dictionaries	Loop through dictionary and invert keys and values.	dict comprehension
Sets	Loop through a set and print each element.	for-loop
Sets	Loop through a set and add only even numbers to a new set.	for-loop + if
Sets	Loop through two sets and print common elements.	for-loop + set intersection
Sets	Loop through a set and count how many elements are greater than 5.	for-loop + if + counter
Sets	Loop through a set and convert all elements to strings.	set comprehension
Tuples	Loop through a tuple and print each item.	for-loop
Tuples	Loop through a list of tuples and print the first item of each.	for-loop + unpacking
Tuples	Loop through a tuple and build a reversed version.	reversed() or slicing
Tuples	Loop through a tuple and return all items that are strings.	for-loop + isinstance
Tuples	Loop through two tuples simultaneously and return the sum of elements.	zip()
