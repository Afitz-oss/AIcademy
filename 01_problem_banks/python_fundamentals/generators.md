| Data Structure                | Problem                            | Suggested Method / Pattern                                     | Example                                                                         |
| ----------------------------- | ---------------------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| List/tuple                    | Sum evens                          | `sum(x for x in xs if x % 2 == 0)`                             | `sum(x for x in nums if x % 2 == 0)`                                            |
| List/tuple                    | Count matches                      | `sum(1 for x in xs if pred(x))`                                | `sum(1 for w in words if w.isalpha())`                                          |
| List/tuple                    | Any match?                         | `any(pred(x) for x in xs)`                                     | `any(x < 0 for x in nums)`                                                      |
| List/tuple                    | All match?                         | `all(pred(x) for x in xs)`                                     | `all(x.islower() for x in tokens)`                                              |
| List/tuple                    | Min/Max with filter                | `min(x for x in xs if pred(x))`                                | `min(x for x in nums if x > 0)`                                                 |
| List/tuple                    | Argmin-like key                    | `min(xs, key=func)` (not a genexp, but pairs well)             | `min(words, key=len)`                                                           |
| List/tuple                    | Sorted filtered stream             | `sorted(x for x in xs if pred(x))`                             | `sorted(x for x in nums if x%2)`                                                |
| List/tuple                    | Deduplicate to set                 | `set(x for x in xs if pred(x))`                                | `set(w.lower() for w in words)`                                                 |
| List/tuple                    | First matching item                | `next((x for x in xs if pred(x)), default)`                    | `next((u for u in users if u.id==id), None)`                                    |
| List of lists                 | Flatten (1 level)                  | `(y for x in xss for y in x)`                                  | `sum(len(row) for row in rows)`                                                 |
| Strings                       | Count substring in lines           | `sum(1 for line in lines if sub in line)`                      | `sum(1 for ln in f if "error" in ln)`                                           |
| Strings                       | Any token numeric?                 | `any(tok.isdigit() for tok in s.split())`                      | —                                                                               |
| Dict                          | Sum values by condition on keys    | `sum(v for k,v in d.items() if pred(k))`                       | `sum(v for k,v in prices.items() if k.startswith("A"))`                         |
| Dict                          | Any value exceeds                  | `any(v > t for v in d.values())`                               | —                                                                               |
| Dict → invert (unique values) | `dict((v,k) for k,v in d.items())` | —                                                              |                                                                                 |
| File (iterator)               | Count lines matching               | `sum(1 for ln in f if pred(ln))`                               | `sum(1 for ln in f if "sum" in ln)`                                             |
| File (iterator)               | Stream numbers from file           | `sum(int(x) for ln in f for x in ln.split())`                  | —                                                                               |
| Pairwise                      | Adjacent diffs                     | `(b - a for a,b in zip(xs, xs[1:]))`                           | `sum(1 for a,b in zip(s,s[1:]) if a==b)`                                        |
| Windowed sum (size k)         | Sliding windows via zip            | `(sum(window) for window in zip(*[xs[i:] for i in range(k)]))` | `max(sum(w) for w in zip(*[xs[i:] for i in range(3)]))`                         |
| Cartesian filter              | Filtered product                   | `((a,b) for a in A for b in B if pred(a,b))`                   | `sum(1 for a,b in product if a<b)`                                              |
| Index + value                 | Find index of first match          | `next((i for i,x in enumerate(xs) if pred(x)), -1)`            | —                                                                               |
| Group-ish count (no lib)      | Count truths                       | `sum(1 for _ in (x for x in xs if pred(x)))`                   | —                                                                               |
| Safe average (filtered)       | Mean of positives                  | `total/ count` with two genexps                                | `s=sum(x for x in xs if x>0); n=sum(1 for x in xs if x>0); avg=s/n if n else 0` |
