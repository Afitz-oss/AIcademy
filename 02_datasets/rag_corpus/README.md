# RAG Corpus

A small set of short documents on AI engineering concepts, used as the knowledge base for the RAG problem bank (`01_problem_banks/ai_engineering/rag.md`).

You'll build a retrieval system over these docs — ingest them, embed them, and ask questions like "what is an embedding?" Your pipeline should retrieve the right document and answer from it.

**Want to use your own documents?** Drop any `.md` or `.txt` files into this folder and re-run your `ingest()` function. The corpus is just a folder of text — swap it for company docs, lecture notes, or anything you want to query.

Each file is intentionally short and self-contained so retrieval boundaries are easy to reason about.
